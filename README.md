# 🚨 RespondNow




| ID | Description |
|----|-------------|
| [] | Any visitor can view all emergency services and active first responders (Police, Firefighters) |
| [] | Any visitor can register and create a personal account |
| [] | First responders can enable live location sharing while on active duty |
| [] | Responder live location is shared with the alerting user until the emergency is resolved |
| [] | Both guest and registered users can trigger an emergency alert broadcast to all available responders |
| [] | Responders currently handling an emergency will not receive new alerts |
| [] | Responders can see the incident location and a list of other responders who accepted the alert |
| [] | Accepted responders receive turn-by-turn routing to the incident location |
| [] | Duplicate alerts within a 500m radius are blocked; user is notified that responders are on the way |
| [] | Multiple alerts within a 500m radius are queued and assigned to nearby responders |
| [] | Responders en route auto-accept queued alerts within the 500m radius and can view all incidents |
| [] | User IP/MAC address is temporarily logged during an active emergency to deter spam |
